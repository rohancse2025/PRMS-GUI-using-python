# Patient Records Management System (PRMS)
A local-first desktop healthcare management application designed to securely manage patient records, perform clinical trend analysis, and provide rule-based decision support. The system operates fully offline using a structured SQLite database and a Python-based graphical interface.
## Overview
## Core Functionalities
1. Clinical Data Management – CRUD operations for patient records.
2. Risk Assessment – Heuristic-based classification (High, Medium, Low).
3. Admission Recommendation – Intelligent decision logic.
4. Data Visualization – Graphical reports and trend analysis.
5. Similarity Detection – Identifies related historical cases.
## Technologies Used

- Python  
- Tkinter  
- SQLite3  
- Matplotlib  
- NumPy
## Installation & Execution

Install dependencies:

pip install matplotlib numpy

Run the application:

python prms_main.py
